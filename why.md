## ⛪ ကက်သလစ်မြန်မာကျမ်းစာ Database Project အတွက် စေတနာရှင် ဝါသနာရှင်များ ဖိတ်ခေါ်ခြင်း

JMJ,
အားလုံးပဲ မင်္ဂလာပါ။

ကျွန်တော်ဟာ နည်းပညာနဲ့ Programming ကို ဝါသနာပါတဲ့ ကက်သလစ်ဘာသာဝင်တစ်ယောက် ဖြစ်ပါတယ်။ အင်တာနက်ပေါ်မှာ ရှာဖွေကြည့်တဲ့အခါ Protestant ကျမ်းစာ App တွေနဲ့ အခြားဘာသာရေး App တွေ အများအပြား ရှိနေပေမဲ့ ကျွန်တော်တို့ **ကက်သလစ်မြန်မာကျမ်းစာ (Catholic Myanmar Bible)** အတွက်တော့ Digital Database နဲ့ Application တွေဟာ အလွန်ပဲ ရှားပါးနေတာကို သတိပြုမိပါတယ်။

ဒါကြောင့်မို့လို့ ဘာသာတူတွေ အလွယ်တကူ ဖတ်ရှုနိုင်ဖို့နဲ့ နောင်တစ်ချိန်မှာ Developer တွေက iOS, Android နဲ့ Web Application တွေမှာ အခမဲ့ ထည့်သွင်းအသုံးပြုနိုင်မဲ့ **Open-source Database** တစ်ခု ဖြစ်ပေါ်လာစေဖို့အတွက် ကျမ်းစာစာသားတွေကို ကူညီရိုက်ပေးမဲ့ စေတနာရှင်တွေကို ဖိတ်ခေါ်လိုပါတယ်။

### ✨ ဘာကြောင့် ပါဝင်သင့်သလဲ?

* **ဓမ္မကုသိုလ် ရရှိခြင်း:** ကျမ်းစာကို ဖတ်ရှုရင်း ကွန်ပျူတာမှာ ရိုက်ကူးခြင်းဖြင့် ကျမ်းစာလာစကားတော်တွေကို ပိုမိုမှတ်မိစေပြီး သာသနာပြုရာလည်း ရောက်ပါတယ်။
* **Digital သာသနာပြုခြင်း:** ကျွန်တော်တို့ စုပေါင်းရိုက်ကူးလိုက်တဲ့ စာသားတွေဟာ နောင်တစ်ချိန်မှာ မြန်မာကက်သလစ်ဘာသာဝင်တွေအတွက် အဖိုးမဖြတ်နိုင်တဲ့ Digital အမွေအနှစ် ဖြစ်လာမှာပါ။
* **အသိအမှတ်ပြု ဂုဏ်ပြုခြင်း:** ပါဝင်ကူညီပေးကြသူတစ်ဦးချင်းစီရဲ့ အမည်ကိုလည်း သက်ဆိုင်ရာ ကျမ်းအလိုက် ဂုဏ်ပြုမှတ်တမ်းတင် ပေးသွားမှာ ဖြစ်ပါတယ်။
* *ဥပမာ - ကမ္ဘာဦးကျမ်း (မောင်ထွန်းထွန်း)၊ ထွက်မြောက်ရာကျမ်း (Aung Aung & Friends Group)*



### 🛠 ကျွန်တော်တို့ ဘာလုပ်ကြမှာလဲ?

ကျွန်တော်ကိုယ်တိုင်က Developer တစ်ယောက်ဖြစ်တဲ့အတွက် Database ရလာတာနဲ့ App ရေးသားဖို့ အဆင်သင့်ရှိပါတယ်။ လက်ရှိမှာ စာသားတွေ (Data Entry) လိုက်ကူးရေးဖို့အတွက်ပဲ လူအင်အား လိုအပ်နေတာပါ။ ကျမ်းစာစာအုပ် တစ်အုပ်လုံးကို တစ်ယောက်တည်း ရေးဖို့ဆိုတာ မလွယ်ကူပေမဲ့ ကျွန်တော်တို့ အစုအဖွဲ့နဲ့ ဝိုင်းဝန်းလုပ်ဆောင်ကြမယ်ဆိုရင် မကြာခင်မှာ အောင်မြင်နိုင်မှာပါ။

> **"တစ်ယောက်အားနဲ့ မရနိုင်တာကို အများအားနဲ့ အကောင်အထည်ဖော်ကြရအောင်ပါ"**

### 📱 နမူနာအနေနဲ့

ကျွန်တော် ၂၀၂၅ ခုနှစ်မှာ YBS App (DD BitePu) ကို ကိုယ်တိုင်ရေးသားခဲ့ဖူးပါတယ်။ အစကတော့ ဘယ်သူမှ မလုပ်သေးလို့ စောင့်နေခဲ့ပေမဲ့ ကိုယ်တိုင် စလုပ်လိုက်တဲ့အခါ နောက်ပိုင်းမှာ YBS App တွေ အများကြီး ပေါ်ထွက်လာတာကို တွေ့ရပါတယ်။ ဒီလိုပဲ ကက်သလစ်ကျမ်းစာ App ကိုလည်း ကျွန်တော်က စတင်လိုက်မယ်ဆိုရင် နောက်ပိုင်းမှာ တခြားAppတွေ အများကြီး ပေါ်ထွက်လာပြီး ဘာသာတူတွေအတွက် အဆင်ပြေသွားမယ်လို့ ယုံကြည်ပါတယ်။
